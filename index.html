March 18th, 2015 Boxes!!

Content Usually the image or the text, in the very middle of the "box".
Padding Clears area around content, padding is transparent.
Border A border that goes around the padding and content.
Margin Clears an area outside the border, The margin is transparent.

Boxing Sizing

- HTML elements are boxes: four elements inside it (content, padding, border, margin) - Sizing a box correctly can be difficult.

Two techniques you can use are as follows:

-Set pixels in percents (%) rather than pixels (px)
-Set box-sizing as border-box for every element: { box-sizing: border-box; } (obvious CSS formatting would be applied)

Box Positioning

Divs are block elements (NOT INLINE)
- By default this makes them take up the entire width of a page
- So, add the rule "display:flex;" to appropriate CSS. This will override this behavior and let divs appear next to each other.
